Часть 2

Программирование движений

Шаг 1.
Робот управляется через Wi-Fi антенну и специальную среду разработки с командной строкой на компьютере. Всё просто: надо задать нужную команду для микроконтроллера.

Сначала запустим робота и все системы. Выберите правильную команду, чтобы это сделать:

robot3/system: ?
  • [view data]
  • [run]
  • [systemReset]

Активирую систему...
Питание подключено
Нейросеть активирована
Робот готов к работе!

Шаг 2.
Роботу нужно уметь двигаться — давайте проверим, готов ли робот к движению.

robot3/system: ?
  • [vision]
  • [init]
  • [move]

Проверка сервоприводов
Электродвигатели готовы к работе
Устойчивость: удовлетворительная

Шаг 3.
Давайте проверим камеру и систему компьютерного зрения робота. Выберите подходящую команду.

robot3/system: ?
  • [vision]
  • [imgWidth]
  • [run]

Ширина получаемого изображения:
imgWidth = 720
Высота получаемого изображения:
imgHeight = 540
Частота кадров:
36 FPS
Система визуального определения объектов активирована: вижу мяч

Шаг 4.
Отлично! Робот распознал мяч — он лежит прямо у него под ногами. А какой робофутбол без голевых моментов? Пора бить по мячу! Выберите подходящую команду.

robot3/system: ?
  • [visionOff]
  • [kick]
  • [systemReset]
